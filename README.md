# UPBGE-LowPolyWater-Component

This component is designed to help UPBGE game developers, this component is used to move the mesh vertices of an object and create a water effect. Easy to implement!

<img src="https://i.imgur.com/rdUrwoU.png"/>

# How to implement?
Create a plan and make some subdivisions.

<img src="https://i.imgur.com/k5zcGbh.png"/>

Add the script and register the component: LowpolyWater.Water

<img src="https://i.imgur.com/y9dyOTh.png"/>

Set the values to your liking and it's ready to use!
